# EB Fonts

> EB fonts for SCSS and CSS

## Installation

```bash
npm install ebudvikling/eb-fonts --save
```

## Usage

Variables and maps:

```scss
@import "node_modules/eb-fonts/src/all.scss";
```

Only size portion

from xxsmall to xxxlarge

```scss
@import "node_modules/eb-fonts/src/sizes/all.scss";
```
